# get-it-done

## Quick start:

1. Install node modules  
    ```sh
      npm install
    ```
2. Build a bundle.js 
    ```sh
      npm run build
    ``` 
3. Start the server
    ```sh
      npm start
    ``` 
4. App is running at
    ```sh
      http://localhost:3535
    ```
    
5. For developer mode... (webpack dev server is set up)
    ```sh
      npm run dev
    ```
